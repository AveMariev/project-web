Это приложение создает WEB - страницу, на котором будут находиться 3 графика: с курсом доллара и евро к рублю, с количеством населения в России и с ВВП США И России в долларах США.
Возле каждого графика в верхнем правом углу будут находиться дополнительные инструменты для работы с графиком: Приближение и отдаление крафика; кнопка, переключающая курсор в режим перемещения по графику; кнопка, переключающая курсор в 
режим, позволяющий выделять необходимое место на графике; Сохранение графика в виде PNG файла; Также, двойным нажатием на график левой кнопкой мыши или при нажатии на последнюю кнопку график возвращается в изначальное положение.
Также, при наведении мыши на точку на графике можно увидеть информацию возле курсора по этой точке. График курса валют обновляется автоматически, остальные графики обновляются раз в год вручную.
